# Design System

## Principles
- Minimal
- One card style
- No gradients by default
- No dashboard chrome unless strictly necessary
- Low-noise surfaces, strong spacing, clear typography

## Spacing
- `--space-1`: 4px
- `--space-2`: 8px
- `--space-3`: 12px
- `--space-4`: 16px
- `--space-5`: 20px
- `--space-6`: 24px
- `--space-7`: 32px
- `--space-8`: 40px
- `--space-9`: 48px

## Typography
- `--text-display`: large hero/display
- `--text-h1`: primary page title
- `--text-h2`: section title
- `--text-h3`: card/dialog title
- `--text-body-lg`: prominent body text
- `--text-body`: default body text
- `--text-body-sm`: supporting body
- `--text-label`: labels, pills, microcopy

## Colors
- `--color-primary`: main dark ink
- `--color-primary-strong`: stronger dark state
- `--color-accent`: secondary accent
- `--color-accent-strong`: alert/accent red
- `--color-bg`: app background
- `--color-surface`: cards, sheets, inputs
- `--color-surface-muted`: subtle raised neutral
- `--color-text`: main text
- `--color-text-muted`: secondary text
- `--color-border`: default border
- `--color-border-strong`: emphasized border
- `--color-focus`: focus ring

## Buttons
- `.ds-btn`: primary
- `.ds-btn-secondary`: standard outlined/neutral
- `.ds-btn-ghost`: text/ghost

## Card Pattern
- `.ds-card`
- single surface, single border, single shadow
- rounded with `--radius-lg`

## Inputs
- standard `input`, `textarea`, `select`
- white surface
- single border
- consistent focus ring

## Layout
- `.ds-container`
- `--container-max`: 760px
- `--page-padding`: 16px
